# CSC-550-Fundamentals-of-Mobile-Apps-Dev
Fall 2025

Android
1. ```cd choe_tiffany_week3-4/TurboModuleExample```
2. have node version 20+
3. run ```npm install```
4. in one terminal start metro bundler ```npm run start```
5. in another terminal build app with ```npm run android```

IOS
1. ```cd choe_tiffany_week3-4/TurboModuleExample```
2. have node version 20+
3. ```cd ios```
4. run ```bundle install```
5. run ```bundle exec pod install```
6. from /TurboModuleExample folder run ```npm install```
7. in one terminal start metro bundler ```npm run start```
8. in another terminal build app with ```npm run ios```
